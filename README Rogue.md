##Описание
Данный проект является минимальной сборкой файлов, необходимых для создания игры на движке RGSS, независимой от конструктора RPG Maker XP.

##Требования для работы
Для корректной работы проекта требуется RGSS102E.dll.

Полный комплект вы можете найти по ссылке: `https://yadi.sk/d/yHem98DJX6Pe4`

Данный проект не требует наличия установленного RPG Maker XP!

##Исходный код
Весь исходный код проекта находится в файлах в папке lib.

Если вам срочно необходимо отредактировать ядро этого проекта:
1. Создайте файл проекта Game.rxproj
2. Внесите в него содержимое: RPGXP 1.02
3. Сохраните его
4. Скачайте Gemini от Forever Zero по одной из ссылок:

* http://forum.chaos-project.com/index.php?topic=10420.0
* http://rmrk.net/index.php?topic=43771.0

##Откройте только что созданный файл проекта в Gemini

Для записи в файл flow.log используйте глобальную переменную $log или же метод wr, который выводит сообщение в консоль и файл одновременно.

##Очередь выполнения кода
В данной версии исходные файлы исполняются в алфавитно-цифровом порядке.

Индексы:
---
0 - файл инициализации библитек, содержит базовый класс Main

1-99 - зависимости; код, без которого все последующее работать не будет

100-199 - классы игровых данных

200-499 - классы интерфейса

500-998 - классы сцен

999 - файл модификации класса Main для запуска модулей приложения
